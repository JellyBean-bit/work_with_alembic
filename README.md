# Запуск
1. Необходимо склонировать себе репозиторий. 

2. Файл с перменными окружениями уже есть (```.env```)
3. Запустить контейнер с помощью команды:
   ```
   docker compose up --build  
   ```
4. После запуска контейнера, необходимо применить миграции командой:
   ```
   alembic upgrade head
   ```
   Миграции должны быть успешно применены

3. В терминале находясь в корне проекта прописать:
`uv run python main.py`

**Проект запущен!**


# Тесты

1. Запуск тестов слоя репозиториев.

   В корне проекта прописать: 
   ```
   uv run python -m pytest tests/test_repositories/ -v
   ```
2. Запуск тестов слоя сервисов.

   В корне проекта прописать:
   ```
   uv run python -m pytest tests/test_services/ -v
   ```
